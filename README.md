## CycloneDDS examples 

This repository is created for the [cdds](https://www.github.com/atolab/cdds) examples. 

### How to compile and install cdds 

#### prerequisite

``Java8, cmake, gcc``

```bash
$> git clone https://www.github.com/atolab/cdds ~/cdds
$> cd ~/cdds
$> mkdir build
$> cd build 
$> cmake ../src
$> make
$> make install
```
